## 주요 내용  

### 1. 로그인 및 회원가입

- 이메일 & 비밀번호 기반 로그인<br>
- 아이디 찾기: 이메일 인증 후 아이디 제공<br>
- 비밀번호 찾기: 이메일 인증 후 임시 비밀번호 발급<br>
- 회원가입: 3단계 진행 (정보 입력 → 이메일 인증 → 프로필 설정)<br>
- 아이디/닉네임 중복 검사<br>
- 비밀번호 정규식 검사 (영문+숫자+특수문자 8자 이상)<br>
- 주소 검색 API 활용<br>
- 프로필 사진 업로드<br>

### 2. 검색 기능
- 키워드 검색: 제목, 설명, 카테고리 기반 필터링<br>
- 이미지 검색: 이미지 업로드 시 배경 제거 및 YOLOv8 기반 카테고리/이름 분류 (FastAPI 서버를 활용하여 처리)<br>

### 3. 상품 등록
- 필수 입력 정보: 제목, 상품명, 이미지, 카테고리, 대여 가격, 대여 가능 기간, 설명<br>

- 유사 상품 가격 비교: 같은 카테고리 내 가격 분포 그래프 제공<br>

### 4. 거래 및 대여 시스템
- 상세 페이지: 상품 정보, 이미지, 동영상 확인<br>
- 채팅 기능: 실시간 메시지 & 중간 거래 장소 추천 기능<br>
- 대여 신청 프로세스: 신청 후 대여자가 "빌려주기" 승인 시 대여 확정 / 거래 종료 후 반납 확인 버튼 클릭 시 종료<br>

### 5. 숏폼 페이지
- 상품 홍보 영상 업로드 가능<br>
- 랜덤 재생 및 자동 소리 ON/OFF 기능 제공<br>

### 6. 중간거리 찾기
- 판매자 & 대여자의 주소 입력 시 자동 중간 지점 계산 후 추천<br>

### 7. 마이페이지 및 리뷰 시스템
- 내 정보 변경 (비밀번호, 주소, 닉네임, 프로필 사진 변경 가능)<br>
- 거래 내역 조회 및 리뷰 작성-별점 평가<br>

## 인공지능
### 1. YOLOv8 모델
YOLOv8은 객체 검출 모델로, 실시간 객체 인식 및 분류에 최적화된 격량 네트워크입니다. 본 프로젝트에서는 제품 등록 시 이미지 내의 객체를 신속하게 탐지하고 자동으로 분류하는데 사용됩니다. 앵커 프리(anchor-free) 방식을 채택하여 복잡한 배경에서도 안정적인 인식 성능을 보이고 빠른 추론 속도와 높은 정확도를 통해 다양한 제품 카테고리를 분류하는 데 적용할 수 있습니다.

### 2. ResNet50 모델
ResNet50은 50개의 층으로 구성된 심층 잔차 네트워크로, 이미지 분류와 특징 추출 분야에서 강력한 성능을 가집니다. 잔차 연결(Residual Connection)을 통해 매우 깊은 네트워크에서도 효과적으로 학습할 수 있고 제품 이미지의 세밀한 특징을 추출할 수 있습니다. 본 모델은 사용자가 업로드한 제품 이미지와 등록된 상품의 이미지 간 유사도 비교에 활용되며, 유사 이미지 검색 기능을 구현하는 핵심 기능입니다.

### 3. 크롤링

본 프로젝트에서는 Selenium을 활용하여 웹 크롤링을 수행했습니다. Selenium은 동적 웹페이지에서도 자동으로 데이터를 수집할 수 있는 라이브러리로, 본 프로젝트에서는 쿠팡의 제품 이미지를 크롤링하는 데 사용했습니다.

### 4. 데이터셋
본 프로젝트에서는 Roboflow 사이트를 활용하여 크롤링한 쿠팡 제품 이미지의 전처리를 진행했습니다. 이미지 라벨링과 이미지 증강 등의 과정을 거쳐 서 학습 데이터를 생성했습니다. 이 과정을 통해 모델의 일반화 능력을 높이고, 실제 서비스 환경에서 높은 인식 성능을 기대할 수 있습니다.

# 🛠 결론 🏗

본 프로젝트에서는 기존의 B2C 렌트와 중고 물품 판매를 결합한 중고 물품 대여 플랫폼인 ‘대여돼요’를 개발했습니다. 본 플랫폼은 기존 렌트 서비스와 차별된 C2C 기반의 대여 서비스를 제공하여 사용자 간의 자원 공유와 활용성을 극대화하는 것을 목표로 정했습니다. 이를 통해 기존 플랫폼이 가지는 한계를 극복하고 사용자 중심의 대여 서비스를 개발했습니다.

해당 서비스는 딥러닝 모델인 YOLOv8과 ResNet50을 활용하여 다양한 사용자 편의 기능을 구현했습니다. 사용자가 빌려주고 싶은 상품을 등록할 때 YOLOv8을 통해 자동으로 카테고리와 이름을 분류함으로써 상품 등록 과정의 편의성을 높였고 상품 등록 시 유사한 상품들의 가격 정보를 차트로 제공함으로써 합리적인 가격 설정을 할 수 있도록 서비스를 제공했습니다. 또한, ResNet50을 이용하여 사용자가 원하는 상품의 이미지를 입력하면 가장 유사한 상품을 검색할 수 있는 기능을 제공해 사용자 경험을 향상시켰습니다. 이와 더불어, 숏폼 콘텐츠를 활용한 상품 홍보와 설명 기능은 사용자들에게 더 흥미롭고 직관적인 정보를 제공하여 플랫폼의 활용도를 높이는데 기여하고 있습니다.

‘대여돼요’ 서비스는 사용자간 신뢰를 바탕으로 한 C2C 렌트 서비스를 통해 자원의 재사용과 공유 경제를 실현할 수 있는 플랫폼으로 자리 잡을 것으로 기대됩니다. 
향후 연구에서는 데이터 기반 서비스를 더욱 향상시키고, 클라우드 기반 인프라를 구축함으로써 서비스의 안정성과 확장성을 강화할 계획입니다. 이를 통해 본 플랫폼이 사용자에게 더욱 신뢰할 수 있고 편리한 서비스를 제공하며, 지속 가능한 중고 렌트 플랫폼으로 성장할 수 있을 것으로 기대됩니다.
